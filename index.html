This neural net is based off one I built in Python (and currently rewriting in C++) which performs the full test, train, and output cycle. The code for this page simply takes the weights and biases from the full implementation and gives an output for a given input. The neural net in this page does not learn from its successes or failures.

This page works first by taking the image drawn in the sketch canvas and resizing it to the appropriate size for the neural net and applying a pure white background. Then a slight image blur is applied to achieve an anti-aliasing effect in an effort to more closely match the look of the original training data. Next, the data for each pixel is extracted and transformed to the correct range where a 0.0 is white and a 1.0 is black with shades of grey in between. The result is a 784 number array which is fed to the neural net.
